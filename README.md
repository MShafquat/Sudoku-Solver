# Sudoku Solver

This program solves a sudoku puzzle using backtracking algorithm.
Usage: `main.py [-h] [-o OUTPUT_FILE] input_file`.
Input file must be a 9x9 grid of integers from
0 to 9, where 0 represents cells to be solved.
Example input file:

```text
9 0 4 0 1 2 0 3 0
0 7 1 0 0 0 0 0 0
8 3 0 0 0 4 5 6 0
0 0 6 2 5 0 0 0 0
0 8 5 9 0 3 6 4 0
0 0 0 0 4 6 1 0 0
0 9 7 1 0 0 0 8 6
0 0 0 0 0 0 9 2 0
0 5 0 4 2 0 3 0 7
```
